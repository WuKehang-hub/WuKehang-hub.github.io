## 快速开始

```bash
# 安装依赖
npm install

# 本地预览
npx hexo server

# 新建文章
npx hexo new "文章标题"

# 生成静态文件
npx hexo generate

# 部署到 GitHub Pages
npx hexo deploy
```

## 项目结构

```
.
├── _config.yml          # Hexo 主配置
├── _config.anzhiyu.yml  # Anzhiyu 主题配置
├── source/              # 文章 & 页面源文件
│   └── _posts/          # 博客文章
├── themes/              # 主题目录
└── public/              # 生成的静态文件
```
